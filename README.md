## 🛠️ Observatory Architecture

- **Framework**: Expo SDK 54 (Managed Workflow) — deploys flawlessly across iOS, Android, and Web  
- **Language**: TypeScript  
- **Navigation**: React Navigation (Native Stack + Bottom Tabs)  
- **Data Core**:  
  - Local — `@react-native-async-storage/async-storage` + `expo-secure-store`  
  - Optional Sync — Supabase hyperspace link  
- **Visualization Array**: Victory Native — charts that feel alive  
- **Support Systems**:  
  - Notifications (`expo-notifications`)  
  - Print & Share (`expo-print` + `expo-sharing`)  
  - Device Telemetry (`expo-device`, `expo-constants`)  

**Core**: React 19 + React Native 0.81
